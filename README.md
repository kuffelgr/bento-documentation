# Bento Documentation

This repository contains documentation, operational guides, and AI workflow knowledge bases for the **Bento Framework** — the shared platform underlying NCI's Cancer Research Data Commons (CRDC) data commons projects, including ICDC and CTDC.

## Contents

| Path | Description |
|------|-------------|
| `claude/SKILL.md` | Bento Project Command Center skill file — SOPs, JQL recipes, Jira quirks, stakeholder doc standards, and operational context for Claude AI workflows |
| `claude/decisions/` | Scope and architecture decision records — rationale for key choices made during the project |
| `claude/conventions/` | Team conventions and workflow standards Claude applies automatically |
| `claude/epics/` | Claude-optimized briefings for active Bento epics |
| `docs/` | Generated project documents (reference guides, meeting outputs, stakeholder artifacts) |
